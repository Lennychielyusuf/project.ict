Workshop 1 ICT

In de eerste workshop ICT zijn we geïntroduceerd met HTML en CSS. HTML staat voor HyperTekst Markup Language en CSS voor Cascading style sheets. HTML wordt gebruikt om de structuur van een website te maken, CSS wordt dan gebruikt voor de opmaak van deze site. We maakten onze eigen webpaginas aan en leerden hoe we die moeten ontwerpen.
